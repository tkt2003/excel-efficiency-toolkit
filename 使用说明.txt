老头表格助手使用说明

使用前建议：
本工具为本地 Excel 效率工具，不会主动上传用户文件。处理正式文件前，建议先自行备份；涉及批量删除、批量重命名、批量替换链接等操作时，请先用测试文件确认效果。请从作者提供的固定链接获取发布包，避免使用来源不明的 exe 文件。

一、软件说明

老头表格助手用于处理常见 Excel 批量整理工作，适合审计、财务、报表整理、底稿维护等场景。

常用功能可以直接在软件主界面看到。第一次使用时，建议先用测试文件或文件副本试跑一遍，确认效果后再处理正式文件。

新版主界面采用左侧分组导航和功能卡片布局，功能名称与原有按钮保持一致。

二、运行环境

1. 建议环境：Windows 系统 + Microsoft Excel。
2. 普通用户直接双击“老头表格助手.exe”运行。
3. 如果发布包中有“使用说明.txt”，建议和“老头表格助手.exe”放在同一个文件夹。
4. 部分功能需要调用已安装的 Microsoft Excel，处理文件时建议关闭无关工作簿。

三、使用前建议

1. 正式处理重要文件前，建议先复制一份副本试跑。
2. 批量删除、批量重命名、按颜色清空、模板批量生成等功能会修改文件内容或文件名，执行前请确认文件已备份。
3. 部分功能会直接保存处理结果，具体以弹窗和运行日志提示为准。
4. 写入当前打开工作簿的功能，多数不会自动保存，检查无误后请自行保存。
5. .xls 老格式在部分功能中不支持，建议先另存为 .xlsx 或 .xlsm。
6. 如果功能生成规则表，请先填写、保存并关闭规则表，再回到工具继续执行。

四、外部链接文件说明

部分报表模板可能包含外部链接。处理这类文件前，建议先用副本试跑，并在处理后打开文件检查链接是否正常。

当前“按颜色清空内容”功能已经对外部链接文件做了保护：发现目标文件包含 externalLinks 时，会使用 Excel 方式清空并保存，避免直接用普通方式保存外链文件。

“按模板批量生成 Excel”“批量更换多文件链接”等功能本身会处理外部链接，执行前请确认模板和链接源文件路径正确。

五、功能说明

1. 一簿按工作表拆分为多个文件

用途：
把一个 Excel 工作簿中的每个工作表分别导出为独立文件。

操作：
点击“一簿按工作表拆分为多个文件”，选择源 Excel 文件，再选择输出文件夹。

保存和备份：
不修改、不保存源文件；只在输出目录生成新文件；不生成备份。

注意：
输出目录已有同名文件时，会自动追加序号，不覆盖原文件。


2. 多表合并到一个新表

用途：
把同一个工作簿中多个可见工作表的数据追加合并到一个新工作表。

操作：
点击“多表合并到一个新表”，选择源 Excel 文件，按提示输入结果 sheet 名、表头行号、数据起始行号。

保存和备份：
会在所选工作簿中新增结果工作表；不自动保存；不生成备份。

注意：
目录页、说明页、辅助页如果格式像数据表，也可能被合并进去，处理前请确认工作簿内容。


3. 多簿到一簿

用途：
把多个 Excel 文件中的指定工作表导入到一个已有目标工作簿。

操作：
点击“多簿到一簿”，选择多个源文件，再选择目标工作簿，输入要导入的 sheet 名；留空时使用每个源文件的第一个可见工作表。

保存和备份：
不修改源文件；会修改并保存目标工作簿；会在目标工作簿同目录生成备份。

注意：
.xls 文件会跳过，建议另存为 .xlsx 或 .xlsm。图片、图表、控件、数据透视表、宏模块等复杂对象不保证完整复制。


4. 按指定列拆分工作表

用途：
按某一列的不同值，把一个工作表拆成多个新工作表。

操作：
点击“按指定列拆分工作表”，选择源 Excel 文件，输入源 sheet 名、拆分列、表头行号、数据起始行号。

保存和备份：
会在所选工作簿中新增工作表；不自动保存；不生成备份。

注意：
拆分列中的空值会归入“空白”工作表。生成的 sheet 名如果重复，会自动追加序号。


5. 生成带链接的工作表目录

用途：
生成“工作表目录”，目录中的工作表名称可以点击跳转。

操作：
点击“生成带链接的工作表目录”，选择源 Excel 文件。

保存和备份：
会新增或复用“工作表目录”工作表；不自动保存；不生成备份。

注意：
目录不包含“工作表目录”自己。


6. 批量删除工作表

用途：
按规则批量删除多个 Excel 文件中的工作表。

操作：
点击“批量删除工作表”，选择需要处理的 Excel 文件。程序会生成规则表，在规则表中填写保留或删除规则，保存并关闭规则表后回到工具确认执行。

保存和备份：
会直接修改并保存原 Excel 文件；执行前请确认文件已备份。

注意：
建议先用副本试跑，确认规则无误后再处理正式文件。程序不会让工作簿被删到 0 个工作表。


7. 批量更换多文件链接

用途：
批量替换多个已有 Excel 工作簿中的外部工作簿链接。

操作：
点击“批量更换多文件链接”，选择目标工作簿。程序生成规则表后，在规则表中填写新链接路径，保存并关闭规则表，再回到工具确认执行。

保存和备份：
会修改并保存所选目标工作簿；默认会先生成备份。

注意：
本功能处理 Excel 外部工作簿链接，不处理 VBA、Power Query、数据透视表连接、ODBC/OLEDB 等复杂连接。


8. 按颜色汇总求和

用途：
根据目标工作簿中的填充颜色，在多个源文件中按相同位置汇总求和。

操作：
在 Excel 中打开目标工作簿，选中一个带目标填充色的单元格，点击“按颜色汇总求和”，再按提示选择汇总范围、源文件和写入方式。

保存和备份：
不修改源文件；会修改当前打开的目标工作簿；不自动保存；不生成备份。

注意：
如果刚修改过目标工作簿的颜色、sheet 结构或汇总区域，请先保存目标工作簿。公式单元格读取的是 Excel 最近一次保存后的计算结果。


9. 按颜色清空内容

用途：
按当前选中单元格的填充颜色，清空同色单元格中的值和公式，保留格式。

操作：
在 Excel 中选中一个带目标填充色的单元格，点击“按颜色清空内容”，选择“当前活动工作簿”或“多个工作簿”，再按提示确认处理范围。

保存和备份：
当前活动工作簿模式会修改当前打开的工作簿，但不自动保存。多个工作簿模式会修改并保存选择的目标文件。

多个工作簿模式默认会先扫描，再只为实际会被修改的文件集中生成批次备份文件夹。勾选“已自行备份，本次不再生成备份文件”后，本次不会自动生成备份。

注意：
建议先用副本试跑，确认颜色和范围无误后再处理正式文件。.xls 暂不支持。带 externalLinks 的文件会使用 Excel 方式处理。


10. 批量重命名文件

用途：
按规则表批量重命名文件。

操作：
点击“批量重命名文件”，选择需要重命名的文件。程序生成规则表后，在规则表中填写新文件名和后缀名，保存并关闭规则表，再回到工具确认执行。

保存和备份：
会直接修改文件名；不生成备份。

注意：
执行前请确认文件已备份，并仔细检查规则表内容。修改后缀名只改变文件名，不转换文件格式。程序不覆盖已有文件，遇到同名目标会自动编号或按设置跳过。


11. 批量重命名工作表

用途：
按规则表批量重命名多个 Excel 文件中的工作表。

操作：
点击“批量重命名工作表”，选择需要处理的 Excel 文件。程序生成规则表后，在“重命名清单”中填写新工作表名，保存并关闭规则表，再回到工具确认执行。

保存和备份：
会直接修改并保存所选 Excel 文件；执行前请确认文件已备份。

注意：
工作表名不能超过 31 个字符，不能包含 : \ / ? * [ ]。隐藏工作表默认跳过，目标名冲突时会自动编号。


12. 数据穿透查询

用途：
查询当前选中单元格或连续区域，在单个工作簿多个 sheet 中的值，或在多个文件同名 sheet 中的值。

操作：
在 Excel 中打开基准工作簿，选中一个单元格或连续区域，点击“数据穿透查询”，选择单文件或多文件模式。

保存和备份：
单文件模式会在当前工作簿新增结果 sheet，不自动保存。多文件模式不会修改源文件，会生成并保存独立结果文件。不生成备份。

注意：
只支持单个单元格或单一连续区域，不支持 Ctrl 多选的不连续区域。.xls 建议另存为 .xlsx 或 .xlsm。


13. 按模板批量生成 Excel

用途：
复制模板工作簿，并按规则替换模板中的外部链接，批量生成新的 Excel 文件。

操作：
点击“按模板批量生成 Excel”，选择“替换一个链接生成”或“替换多个链接生成”，再按提示选择模板、链接源文件和输出目录。多链接模式会生成规则表，请检查并填写后再执行。

保存和备份：
不修改原模板文件，不修改链接源文件；会保存生成出来的新 Excel 文件。模板按复制方式生成新文件，执行前仍建议保留模板和源文件备份。

注意：
执行前请确认旧链接、新链接、输出目录和命名规则。输出文件已存在时会自动编号，不覆盖。


14. 选区 ROUND 保留两位

用途：
把当前 Excel 选区中的数值或公式改成 ROUND 保留两位。

操作：
在 Excel 中选中要处理的区域，点击“选区 ROUND 保留两位”。

保存和备份：
会修改当前选区内容；不自动保存当前工作簿；不生成备份。

注意：
空白、文本、错误值和已经是 ROUND 的公式会跳过。该功能只处理当前选区。

六、常见问题

1. 为什么有些功能要求先保存当前工作簿？
因为程序需要读取磁盘上的最新文件内容。如果 Excel 里有未保存修改，磁盘文件和屏幕内容可能不一致。

2. 为什么 .xls 文件经常提示不支持？
.xls 是老格式，建议用 Excel 另存为 .xlsx 或 .xlsm 后再处理。

3. 为什么批量功能要先生成规则表？
规则表用于让用户先检查和确认处理范围，适合批量操作前统一核对。

4. 为什么有些功能不自动保存？
为了方便用户先检查结果。确认无误后，再由用户自己保存。

5. 没有“使用说明.txt”还能用软件吗？
可以。软件内“使用说明”按钮找不到文件时，会显示内置简版说明。

七、建议使用流程

1. 先复制测试文件或备份正式文件。
2. 打开软件，选择对应功能。
3. 如果功能生成规则表，先检查规则表，再保存并关闭。
4. 回到软件确认执行。
5. 查看日志、结果文件和 Excel 内容。
6. 确认无误后，再保存工作簿或交付结果文件。
