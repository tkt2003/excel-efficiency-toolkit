老头表格助手使用说明

一、软件说明

老头表格助手用于处理常见 Excel 批量整理工作，适合审计、财务、报表整理场景。

当前主要功能包括：
1. 一簿按工作表拆分为多个文件
2. 多表合并到一个新表
3. 多簿到一簿
4. 按指定列拆分工作表
5. 生成带链接的工作表目录
6. 批量删除工作表
7. 批量更换多文件链接
8. 按颜色汇总求和
9. 按颜色清空内容
10. 批量重命名文件
11. 批量重命名工作表
12. 数据穿透查询
13. 按模板批量生成 Excel
14. 选区 ROUND 保留两位

二、运行环境

1. 建议环境：Windows 系统 + Microsoft Excel。
2. 普通用户直接双击“老头表格助手.exe”运行。
3. 如果发布包中有“使用说明.txt”，建议和“老头表格助手.exe”放在同一个文件夹。
4. 部分功能需要调用已安装的 Microsoft Excel，请不要同时打开太多无关工作簿。

三、使用前注意事项

1. 正式处理重要文件前，请先复制一份副本试跑。
2. 批量删除、批量重命名、按颜色清空、按模板批量生成等功能风险较高，执行前请确认文件已经备份。
3. 部分功能会直接修改并保存原 Excel 文件，界面和弹窗中会提示，请仔细看清楚再确认。
4. 多数写入当前打开工作簿的功能不会自动保存，检查无误后需要自己在 Excel 中保存。
5. .xls 老格式在部分功能中不支持，建议先另存为 .xlsx 或 .xlsm。
6. 处理文件前，请尽量关闭目标 Excel 文件，避免被 Excel 占用导致保存失败。
7. 如果生成了规则表，请按规则表里的说明填写，保存并关闭规则表后再回到工具执行。
8. 文件名、工作表名不要使用 Excel 或 Windows 不允许的特殊字符。

四、外部链接文件特别说明

1. 带 externalLinks 的 Excel 文件要谨慎处理。
2. 不要随便用 openpyxl 直接保存带外部链接的文件，可能破坏外部链接结构。
3. 当前“按颜色清空内容”功能已经做了外链保护：发现目标文件包含 externalLinks 时，会走 Excel 方式清空并保存，避免 openpyxl 直接保存。
4. 正式处理带外部链接的文件前，建议先用副本试跑，并打开结果文件检查链接是否正常。
5. “按模板批量生成 Excel”“批量更换多文件链接”等功能本身会处理外部链接，执行前请确认模板和链接源文件路径正确。

五、功能说明

1. 一簿按工作表拆分为多个文件

用途：
把一个 Excel 工作簿中的每个工作表分别导出为独立文件。

操作步骤：
1）点击“一簿按工作表拆分为多个文件”。
2）选择源 Excel 文件。
3）选择输出文件夹。
4）查看日志和输出目录。

是否修改原文件：
不修改源文件。

是否自动保存：
不保存源文件，只在输出目录生成新文件。

是否生成备份：
不生成备份。

注意事项：
如果输出目录已有同名文件，程序会自动追加序号，不覆盖原文件。


2. 多表合并到一个新表

用途：
把同一个工作簿中多个可见工作表的数据追加合并到一个新工作表。

操作步骤：
1）点击“多表合并到一个新表”。
2）选择源 Excel 文件。
3）按提示输入结果 sheet 名、表头行号、数据起始行号。
4）在 Excel 中检查新增的合并结果表。

是否修改原文件：
会在所选工作簿中新增结果工作表，但不删除、不覆盖源工作表。

是否自动保存：
不自动保存。检查无误后自行保存。

是否生成备份：
不生成备份。

注意事项：
目录页、说明页、辅助页如果格式像数据表，也可能被合并进去，处理前请确认工作簿内容。


3. 多簿到一簿

用途：
把多个 Excel 文件中的指定工作表导入到一个已有目标工作簿。

操作步骤：
1）点击“多簿到一簿”。
2）选择多个源 Excel 文件。
3）选择一个已有目标工作簿。
4）输入要导入的 sheet 名；留空时使用每个源文件的第一个可见工作表。
5）按提示选择是否只复制值。
6）查看目标工作簿中的导入结果和日志表。

是否修改原文件：
不修改源文件，会修改目标工作簿。

是否自动保存：
目标工作簿会被保存。

是否生成备份：
会在目标工作簿同目录生成备份。

注意事项：
.xls 文件会跳过，建议另存为 .xlsx 或 .xlsm。复杂对象如图片、图表、控件、数据透视表、宏模块等不保证完整复制。


4. 按指定列拆分工作表

用途：
按某一列的不同值，把一个工作表拆成多个新工作表。

操作步骤：
1）点击“按指定列拆分工作表”。
2）选择源 Excel 文件。
3）输入源 sheet 名、拆分列、表头行号、数据起始行号。
4）在 Excel 中检查新生成的工作表。

是否修改原文件：
会在所选工作簿中新增工作表，不删除、不覆盖源工作表。

是否自动保存：
不自动保存。检查无误后自行保存。

是否生成备份：
不生成备份。

注意事项：
拆分列中的空值会归入“空白”工作表。生成的 sheet 名如果重复，会自动追加序号。


5. 生成带链接的工作表目录

用途：
给当前工作簿生成一个“工作表目录”，目录中每个工作表名称可点击跳转。

操作步骤：
1）点击“生成带链接的工作表目录”。
2）选择源 Excel 文件。
3）在 Excel 中检查“工作表目录”。

是否修改原文件：
会新增或复用“工作表目录”工作表。

是否自动保存：
不自动保存。检查无误后自行保存。

是否生成备份：
不生成备份。

注意事项：
目录不包含“工作表目录”自己。


6. 批量删除工作表（高风险）

用途：
按规则批量删除多个 Excel 文件中的工作表。

操作步骤：
1）点击“批量删除工作表”。
2）选择需要处理的 Excel 文件。
3）程序生成并打开临时规则表。
4）在规则表中填写要保留或要删除的工作表名称，也可填写排除文件名关键词。
5）保存并关闭规则表。
6）回到工具，点击确认执行。

是否修改原文件：
会直接修改原 Excel 文件。

是否自动保存：
会自动保存被修改的文件。

是否生成备份：
本功能不强调自动备份，执行前请自行备份。

注意事项：
这是高风险功能。不要直接对唯一原件操作。规则不要同时混乱填写“保留”和“删除”。程序不会让工作簿被删到 0 个工作表。


7. 批量更换多文件链接

用途：
批量替换多个已有 Excel 工作簿中的外部工作簿链接。

操作步骤：
1）点击“批量更换多文件链接”。
2）选择需要处理的目标工作簿。
3）程序生成规则表。
4）在规则表中填写新链接路径，并确认是否执行。
5）保存并关闭规则表。
6）回到工具确认执行。

是否修改原文件：
会修改所选目标工作簿。

是否自动保存：
会自动保存被修改的目标工作簿。

是否生成备份：
默认会先生成备份。

注意事项：
只处理 Excel 外部工作簿链接，不处理 VBA、Power Query、数据透视表连接、ODBC/OLEDB 等复杂连接。


8. 按颜色汇总求和

用途：
根据目标工作簿中的填充颜色，在多个源文件中按相同位置汇总求和。

操作步骤：
1）在 Excel 中打开目标工作簿，并选中一个带目标填充色的单元格。
2）点击“按颜色汇总求和”。
3）选择只汇总当前 sheet，或汇总所有匹配 sheet。
4）选择源 Excel 文件。
5）选择写入公式或只写入汇总数值。
6）在目标工作簿中检查结果。

是否修改原文件：
不修改源文件，会修改当前打开的目标工作簿。

是否自动保存：
不自动保存目标工作簿。检查无误后自行保存。

是否生成备份：
不生成备份。

注意事项：
如果刚修改过目标工作簿的颜色、sheet 结构或汇总区域，请先保存目标工作簿。公式单元格读取的是 Excel 最近一次保存后的计算结果。


9. 按颜色清空内容（高风险）

用途：
按当前选中单元格的填充颜色，清空同色单元格中的值和公式，保留格式。

操作步骤：
1）在 Excel 中选中一个带目标填充色的单元格。
2）点击“按颜色清空内容”。
3）选择“当前活动工作簿”或“多个工作簿”。
4）按提示确认处理范围。
5）查看日志和处理结果。

是否修改原文件：
当前活动工作簿模式会修改当前打开的工作簿；多个工作簿模式会修改选择的目标文件。

是否自动保存：
当前活动工作簿模式不自动保存。多个工作簿模式会保存被修改的文件。

是否生成备份：
多个工作簿模式默认会先扫描，再只为实际会被修改的文件集中生成批次备份文件夹。用户也可以勾选“已自行备份，本次不再生成备份文件”，但风险由用户自行承担。

注意事项：
这是高风险功能。它会清空值和公式，但保留填充色、字体、边框、批注和合并状态。.xls 暂不支持。带 externalLinks 的文件已有外链保护，但正式处理前仍建议用副本试跑。


10. 批量重命名文件（高风险）

用途：
按规则表批量重命名文件。

操作步骤：
1）点击“批量重命名文件”。
2）选择需要重命名的文件。
3）程序生成并打开规则表。
4）在规则表中填写新文件名和后缀名。
5）保存并关闭规则表。
6）回到工具确认执行。

是否修改原文件：
会直接修改文件名。

是否自动保存：
不涉及 Excel 保存，但会立即执行文件重命名。

是否生成备份：
不生成备份。

注意事项：
这是高风险功能。修改后缀名只改变文件名，不转换文件格式。程序不覆盖已有文件，遇到同名目标会自动编号或按设置跳过。


11. 批量重命名工作表（高风险）

用途：
按规则表批量重命名多个 Excel 文件中的工作表。

操作步骤：
1）点击“批量重命名工作表”。
2）选择需要处理的 Excel 文件。
3）程序生成并打开规则表。
4）在“重命名清单”中填写新工作表名。
5）保存并关闭规则表。
6）回到工具确认执行。

是否修改原文件：
会直接修改所选 Excel 文件中的工作表名称。

是否自动保存：
会自动保存被修改的 Excel 文件。

是否生成备份：
不强调自动备份，执行前请自行备份。

注意事项：
这是高风险功能。工作表名不能超过 31 个字符，不能包含 : \ / ? * [ ]。隐藏工作表默认跳过，目标名冲突时会自动编号。


12. 数据穿透查询

用途：
查询当前选中单元格或连续区域，在单个工作簿多个 sheet 中的值，或在多个文件同名 sheet 中的值。

操作步骤：
1）在 Excel 中打开基准工作簿，选中一个单元格或连续区域。
2）点击“数据穿透查询”。
3）选择“单文件数据穿透查询”或“多文件数据穿透查询”。
4）多文件模式下按提示选择源文件。
5）查看生成的结果 sheet 或结果文件。

是否修改原文件：
单文件模式会在当前工作簿新增结果 sheet。多文件模式不会修改源文件，会生成独立结果文件。

是否自动保存：
单文件模式不自动保存当前工作簿。多文件模式会保存新生成的结果文件。

是否生成备份：
不生成备份。

注意事项：
只支持单个单元格或单一连续区域，不支持 Ctrl 多选的不连续区域。.xls 建议另存为 .xlsx 或 .xlsm。


13. 按模板批量生成 Excel（高风险）

用途：
复制模板工作簿，并按规则替换模板中的外部链接，批量生成新的 Excel 文件。

操作步骤：
1）点击“按模板批量生成 Excel”。
2）选择“替换一个链接生成”或“替换多个链接生成”。
3）按提示选择模板、链接源文件、输出目录。
4）多链接模式会生成规则表，请检查并填写后再执行。
5）查看输出文件和处理日志。

是否修改原文件：
不修改原模板文件，不修改链接源文件。会生成新的输出文件。

是否自动保存：
会保存生成出来的新 Excel 文件。

是否生成备份：
不备份模板，因为模板按复制方式生成新文件；但执行前仍建议自行备份模板和源文件。

注意事项：
这是高风险功能，尤其是模板中有多个外部链接时。执行前请确认旧链接、新链接、输出目录和命名规则。输出文件已存在时会自动编号，不覆盖。


14. 选区 ROUND 保留两位

用途：
把当前 Excel 选区中的数值或公式改成 ROUND 保留两位。

操作步骤：
1）在 Excel 中选中要处理的区域。
2）点击“选区 ROUND 保留两位”。
3）检查选区中的公式变化。

是否修改原文件：
会修改当前选区内容。

是否自动保存：
不自动保存当前工作簿。

是否生成备份：
不生成备份。

注意事项：
空白、文本、错误值和已经是 ROUND 的公式会跳过。该功能只处理当前选区，不处理整本工作簿。

六、常见问题

1. 为什么有些功能要求先保存当前工作簿？
因为程序需要读取磁盘上的最新文件内容。如果 Excel 里有未保存修改，磁盘文件和屏幕内容可能不一致。

2. 为什么 .xls 文件经常提示不支持？
.xls 是老格式，很多现代处理方式不稳定。建议用 Excel 另存为 .xlsx 或 .xlsm 后再处理。

3. 为什么批量功能要先生成规则表？
规则表用于让用户先检查和确认处理范围，减少误删、误改、误替换。

4. 为什么有些功能不自动保存？
为了给用户保留检查机会。确认结果无误后，再由用户自己保存。

5. 没有“使用说明.txt”还能用软件吗？
可以。软件内“使用说明”按钮找不到文件时，会显示内置简版说明。

七、建议使用流程

1. 先复制测试文件或备份正式文件。
2. 打开软件，选择对应功能。
3. 如果功能生成规则表，先认真检查规则表，再保存并关闭。
4. 回到软件确认执行。
5. 查看日志、结果文件和 Excel 内容。
6. 确认无误后，再保存工作簿或交付结果文件。
